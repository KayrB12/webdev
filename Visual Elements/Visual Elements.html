Objective

The purpose of this assignment is to practice integrating various media types into a single, styled webpage. You will demonstrate your ability to link an external CSS file and embed images, videos, audio, and external content using an iframe.

Assignment Scenario

You are creating a simple fan page or information hub for your favorite topic. This could be a hobby (like photography or gaming), a favorite musician, a sports team, a movie, or a place you love. Your page will be called "My Media Hub" and will feature different types of media related to your chosen topic.

Requirements

Your final submission must be a single HTML file (index.html) and a separate CSS file (style.css) that together create a complete webpage meeting the criteria below.

Start by creating a new folder in your github repository called "visual elements"

1. General Structure & Styling (20 pts)

  • DOCTYPE and Tags: Your HTML file must start with the <!DOCTYPE html> declaration and include the proper <html>, <head>, and <body> structure.

  • External CSS: You must link to an external CSS file named style.css. All styling for the page must be contained within this file (no inline styles or <style> tags in the HTML).

  • Content: Your page should have a clear title in the <title> tag and at least one <h1> heading that introduces your topic (e.g., "My Media Hub: The World of Astrophotography").

2. Embedded Media (50 pts)

  • Image (10 pts): Embed at least one relevant image using the <img> tag.

    • You must use the src attribute to specify the path to your image file.

    • You must use the alt attribute to provide a text description of the image for accessibility.

  • Video (15 pts): Embed at least one video using the <video> tag.

    • You must include controls so the user can play/pause the video.

    • You should provide a fallback message for very old browsers (e.g., "Your browser does not support the video tag.").

    • Hint: You can use a sample video file from a site like sample-videos.com or use the files directory in your IDE.

  • Audio (15 pts): Embed at least one audio file using the <audio> tag.

    • You must include controls so the user can play/pause the audio.

    • Hint: You can use a sample audio file from a site like sample-videos.com (they have audio too) or use the files directory in your IDE.

  • Iframe (10 pts): Embed at least one external webpage or content using the <iframe> tag.

    • A common and easy choice is to embed a YouTube video. (Right-click a YouTube video and select "Copy embed code").

    • Alternatively, you can embed a Google Map showing a location related to your topic.

3. CSS Styling (20 pts)
In your style.css file, you must apply styles to make your page visually organized. At a minimum, you must include:

  • A background color or image for the page body.

  • A different font family for your headings and paragraph text (e.g., font-family: Arial, sans-serif;).

  • Padding and margins to create space around your content so it isn't crammed together.

  • A border or background color for each media section (image, video, audio, iframe) to visually separate them.

4. Code Quality & Submission (10 pts)

  • Your HTML and CSS code must be neatly indented and easy to read.

  • All file names should be lowercase with no spaces (use hyphens or underscores if needed, e.g., my-image.jpg).

  • Upload the new folder to github and submit a link to the new webpage
